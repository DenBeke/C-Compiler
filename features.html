Base for the grading of the C2P project (2014-2015)

The project is graded on a scale of 20 points. The table below will be used as a starting point for the grading. 

The column ALL describes the basic requirements. If the basic requirements are satisfied, then you will start from a minimum of 10 points.
Extra points can be earned by implementing optional features from the column OPT. 

When you present your work on the exam, the points from the table will be added based on the insights of each individual student.

ALL
These features need to be implemented by all students.

OPT
These features are optional.
Students can earn extra points with these.
